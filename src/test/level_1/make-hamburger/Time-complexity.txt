- 시간 복잡도는 알고리즘을 처리하는데 얼마의 시간이 걸리는지 알려줍니다.
  이런 알고리즘의 시간 복잡도는 주로 빅-오 표기법을 사용하여 나타냅니다.

  Big-O란 알고리즘의 성능을 수학적으로 표련해주는 표기법입니다.
  이를 통해 알고리즘의 시간관 공간 복잡도를 표현할 수 있습니다.
  빅오 표기법은 알고리즘을 처리하는 실제 시간을 표시하는 것이 아닙니다. 빅오 표기법은 데이터나 사용자의 증가율에 따른 알고리즘의 성능을 예측하기 위해 사용합니다. 

  ** O(1): Constant Time
  입력 데이터의 크기에 상관없이 일정한 시간이 걸리는 알고리즘을

  ** O(n): Linear Time
  입력 데이터 크기에 비례해서 처리시간도 늘어나는 알고리즘을 표현
  -> 언제나 데이터와 시간이 같은 비율로 증가하기 때문에 그래프는 직선으로 표기법은

  ** O(n^2) : Quadratic Time
  입력 데이터의 크기의 제곱만큼 처리시간이 걸리는 알고리즘을 표현할 때 사용
